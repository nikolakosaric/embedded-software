#define GET_GPIO_BIT (   x)    (1 << (x % 8))
#define GET_GPIO_PIN (   x)    CAT3(GPIO, x, _PIN_NUM)
#define GET_GPIO_PORT (   x)    (GPIOA0_BASE + (0x1000 * (x/8)))

These macros assume you are using the GPIO number of the pin as the input. Do not use the pin number. Also do not append 0 in front of single digit GPIO numbers. 

    Ex. GPIO2 is PIN_57 so use GET_GPIO_PORT(2)
